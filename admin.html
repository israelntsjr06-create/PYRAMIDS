Portfólio Feed

Fotos para a seção de feed do portfólio

Como adicionar fotos

  1. Coloque o arquivo de imagem dentro da pasta images/feed/
  2. No campo "Caminho", escreva apenas o nome: ex: feed-37.jpg
  3. Preencha o label (nome do cliente/projeto) e clique em Adicionar
  4. Clique em Salvar tudo quando terminar

Portfólio Story

Stories (formato 9:16 vertical)

Como adicionar stories

  1. Coloque a imagem na pasta images/story/
  2. Escreva o nome do arquivo no campo "Caminho": ex: story-10.jpg
  3. Adicione o label e salve

Portfólio Jogador

Artes esportivas / jogadores

Como adicionar artes

  1. Coloque a imagem na pasta images/jogador/
  2. Escreva o nome: ex: jogador-7.jpg
  3. Adicione o label e salve

Vídeos Drone

Vídeos de drone (mp4, mov, webm)

Como adicionar vídeos

  1. Coloque o arquivo de vídeo na pasta images/ (ou subpasta)
  2. Escreva o caminho relativo: ex: video3.mp4
  3. Adicione o label e salve
  4. Dica: vídeos grandes podem ser hospedados no YouTube/Vimeo — cole a URL embed

Vídeos Feed

Vídeos para feed/reels

Como adicionar vídeos

  1. Coloque o arquivo na pasta images/
  2. Escreva o nome: ex: videofeed4.mp4
  3. Adicione o label e salve

Exportar Configuração

Gere o arquivo de configuração do portfólio

Como funciona o sistema

  1. Gerencie seu conteúdo neste painel e clique em Salvar tudo
  2. O site carrega o portfólio automaticamente — as mudanças aparecem na hora
  3. Se quiser fazer backup ou compartilhar a config, use Exportar JSON
  4. Para restaurar uma config salva, use Importar JSON
